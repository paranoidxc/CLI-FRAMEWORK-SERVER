# CLI-FRAMEWORK-SERVER

** 实现一个基于 cli 常驻内存模式的 最小化 http 带个人 orm 和 lib 的服务框架 **

Usage:
- php >= 7.1
- php configure  --enable-pcntl --enable-sockets
- 启动server php server_for_github.php
- 浏览器打开 127.0.0.1:9000/ 或者 127.0.0.1:9000/[a|b|c].php
